The LiveOps Calendar now includes a powerful search feature that lets you quickly find events by name.

## How to Use

### Basic Search
1. Type the event name (or part of it) in the **"Search by event name..."** box
2. Results appear instantly as you type
3. Matching text is highlighted in yellow

## No Results?

If you see "No events found matching [your search]":
- Check your spelling
- Try a shorter/broader search term
- Verify you have events added
- Try searching for single words instead of phrases
